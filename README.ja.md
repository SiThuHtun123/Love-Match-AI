# 💘 Love Match AI

**（あえての）ネタ系AI Webアプリ** —— 自分の写真と最大10人の候補者の写真を
アップロードすると、AIが「相性」をランキング。相性％・面白い理由コメント・
派手な演出付きで結果を発表します。

> ⚠️ **これはネタアプリです。** 本当の恋愛相性は顔からは予測できません。
> ただし数値はすべて **本物のAI顔分析**（MediaPipe Face Mesh）から算出して
> いるので、面白くて、かつ説明可能です。

## 🚀 実行方法

**Python 3.10** が必要です（3.13はAIライブラリに新しすぎます）。

```bash
# 1. 仮想環境を作成
python -m venv venv

# 2. 有効化
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS / Linux

# 3. 依存関係をインストール
pip install -r love_match_ai/requirements.txt

# 4. 実行
python love_match_ai/app.py
```

ブラウザで **http://127.0.0.1:5000** を開きます。SQLiteデータベースは
初回起動時に自動作成されます。APIキーやインターネットは不要——フォントや
JSライブラリはすべて同梱しています。

## 🕹️ 使い方

1. **新規登録／ログイン**（必須——ゲスト利用は不可）。
2. **新しいマッチ** → 自分の写真をアップ、候補者を **1〜10人** 追加
   （1枚につき、はっきりした顔1つ！）。
3. **マッチを探す！** を押す → AIが全員の顔を分析し、相性％・判定・理由付きで
   ランキングを発表。
4. 結果は自動保存：**履歴**（閲覧・名前変更・削除）と **殿堂**
   （自己ベスト＆統計）で確認できます。
5. 言語は右上でいつでも切替：**EN / 日本語**。

## 🧠 AIモデル

- **モデル：** MediaPipe **Face Mesh** —— 顔ランドマーク回帰モデル。1枚の顔から
  **478点の3Dランドマーク** を予測（＋MediaPipe Face Detectionで顔1つチェック）。
  CPUでローカル動作、約0.2秒/枚。
- **入力：** アップロードされた顔写真。
- **予測：** ランドマーク座標 → そこから **14の特徴量** を計算（笑顔率・目の
  開き・眉の高さ・顔型・唇の厚さ・首の傾き・対称性・明るさ・色温度など）。
- **相性スコア：** 決定論的な *ミックス* 計算式——類似性を評価する成分
  （笑顔シンクロ、バイブス調和…）と、差を評価する「正反対も魅力」成分
  （顔型コントラスト、ぬくもりスパーク…）の組み合わせ。同じ写真なら常に同じ
  結果。自分の写真を候補者にすると100%「ツインフレーム🪞」として検出。
- 教科書の画像分類モデルとは **別モデル・別タスク**（分類ではなく
  ランドマーク回帰＋類似度計算）。

## 🔒 ログインしなければ使えない機能（テーブルCRUD以外）

- **AI相性分析（マッチング）** 本体
- **マッチ履歴**（名前変更・削除＝`matches` / `candidates` のCRUD）
- **殿堂** —— 自分の全期間ランキング＆統計

## 🗄️ データベース（SQLite・独自スキーマ）

| テーブル      | 内容                                                             |
|--------------|------------------------------------------------------------------|
| `users`      | id、ユーザー名、パスワードハッシュ、作成日時                        |
| `matches`    | id、user_id、タイトル、自分の名前・写真、最高スコア、作成日時         |
| `candidates` | id、match_id、名前、写真、スコア、順位、判定、理由、内訳             |
